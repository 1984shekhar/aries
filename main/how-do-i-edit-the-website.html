How to edit

You can also click the edit button when viewing the website which is located on the bottom right of each page.

Note that after you've changed the wiki it is immediately rendered on the http://servicemix.org site so you can test it out; then up to an hour or two later it is updated on the Apache site.

How the site works

There are a few special pages; for example the Navigation page is the left hand navigation bar and the QuickLinks is the top right navigation bar.
