The Message Flow MBean is base MBean used for management by different message flow implementations in the Normalized Message Router (NMR). The Message Flow MBean dervies its life cycel manamagement from the Life Cycle MBean.

Attributes
Name type read/write Description
flowType String read-only the name of message flow typed being used (default is seda)
